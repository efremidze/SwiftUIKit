## Installation

### Swift Package Manager

Add the following to your `Package.swift` file:

```swift
dependencies: [
    .package(url: "https://github.com/yourusername/SwiftUIKit.git", from: "1.0.0")
]
```

## Usage

```swift
import SwiftUIKit

struct ContentView: View {
    var body: some View {
        PrimaryButton(title: "Tap Me") {
            // Handle button tap
        }
    }
}
```
